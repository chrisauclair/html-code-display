HTML Code Display

Overview

The Code Display script lets you show selections of source HTML in automatically generated code tags.

Usage

To start, all HTML code that you want to display should be contained in a parent element with a unique id value.

To create a new Code Display instance, create an element with the class code-display. This element should contain two important attributes:

  • data-code-id: value should match the id of the example code
  • data-language: value should be the language name being displayed (e.g., "javascript", "html", etc.)
